# Irregular verbs trainer
*It is a small web app for training irregular english verbs*

*WARNUNG! NOT FINISHED YET*

## Актуальність
При освоєнні часів англійської мови, одним із етапів є вивчення неправильних дієслів, 
яких у цій мові аж до 200 слів. Автоматизація процесу тестування та відслідковування 
прогресу значно полегшує та прискорює процес вивчення.

Inspired by:
- https://lingualeo.com/ru/course/81
- https://www.microsoft.com/en-us/store/p/irregular-verbs-trainer/9wzdncrdcwvq

but this app is **free** and **open-source**

## Technologies stack
Node.js, HTML 5, CSS 3, Bootstrap 3, JS(ESnext), React.js, Webpack.
